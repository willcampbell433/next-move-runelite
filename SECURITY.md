# Security Policy

## Supported version

Security fixes target the latest version on `master` and the version currently
published through the RuneLite Plugin Hub.

## Reporting a vulnerability

Please use GitHub's **Report a vulnerability** button on the repository's
Security tab. Do not open a public issue for a suspected vulnerability.

Include the affected version, reproduction steps, expected impact, and any
suggested mitigation. You can expect an initial response within seven days.
Confirmed issues will be coordinated privately until a fix is available.

For ordinary bugs and feature requests, use the public issue templates instead.
